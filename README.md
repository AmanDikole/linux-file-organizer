# Automated File Organizer using Bash & Cron

A Linux automation project that organizes files based on file type using Bash scripting and cron jobs.

## Features
- Organizes files into folders (Images, Videos, Music, Documents, Archives, etc.)
- Uses Bash functions for clean and reusable code
- Automatically creates directories if missing
- Prevents overwriting files
- Can be scheduled using cron for continuous automation

## Technologies Used
- Bash Shell Scripting
- Linux
- Cron Jobs
- find, mv, mkdir commands

## How to Run
```bash
chmod +x auto_organize.sh
./auto_organize.sh
